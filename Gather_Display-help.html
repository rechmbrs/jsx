GATHER DISPLAY - HELP
RONC's Photoshop Scripts
by Ron Chambers


USAGE PARAMETERS
SEPARATOR - (YES/no) - YES means to place a vertical line between gathers whereas no means to omit separator line.

SAMPLING - (ALL/periodic) - ALL means to display all gathers whereas with periodic every nth gather is displayed where n = number of images (layers) + 1 or 0 whether the separator is chosen or not.

DISPLAY
The output should be displayed at 100% or larger on LCD screens.

SAVE TO FILE
If saving to a file, it is highly recommended that the file format be not use lossy compression. TIF format is probably the best.
